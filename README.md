# バーキン在庫検索サイト（GitHub Pages版）

エルメスのバーキンバッグの在庫を複数のショッピングサイトから検索できるウェブアプリケーションです。

## 機能

- **複数サイト対応**: Yahoo!ショッピング、楽天市場、メルカリ、eBay、ヤフオクから一括検索
- **フィルタリング**: キーワード、サイズ、タイプ、サイト、色、価格帯で絞り込み
- **お気に入り機能**: 気になった商品をお気に入りに登録
- **価格変動通知**: お気に入り商品の価格が変わった際にバナーで通知
- **レスポンシブデザイン**: スマートフォン・タブレット・PCに対応
- **スマートフォン2列表示**: スマートフォンで1列/2列の切り替え可能

## 自動更新

このサイトは以下のスケジュールで自動更新されます：

- **自動更新**: 毎時間（1時間ごと）
- **手動更新**: ページ上の「今すぐ更新」ボタンで即座に実行

## 開発環境でのセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

このサイトはGitHub Pagesで自動デプロイされます。`main`ブランチへのプッシュで自動的にデプロイが実行されます。

## スクレイピング

スクレイピングはGitHub Actionsで自動実行されます。詳細は`.github/workflows/`を参照してください。
